# 💕 WarmTipPopups - 温馨提示弹窗

一个基于 Python tkinter 的创意弹窗程序，采用**莫兰迪配色**，支持动画效果、主题定制、粒子特效、自定义提示语导入和精美的全屏结束卡片。适合用于表白、创意展示、节日祝福等场景。

## ✨ 功能特性

### 🎬 动画效果
- 弹窗淡入淡出动画
- 弹窗在屏幕上自由移动（带边界反弹效果）
- 舒缓的弹出节奏，不会过于突兀

### 🎨 莫兰迪主题系统
- **💕 浪漫主题** - 灰粉/脏粉色系 + 爱心粒子
- **🌸 清新主题** - 薄荷灰/灰绿色系 + 花瓣粒子
- **✨ 神秘主题** - 薰衣草灰/灰紫色系 + 星星粒子
- **☀️ 温暖主题** - 杏色/脏橘色系 + 爱心粒子
- **🌊 海洋主题** - 雾霾蓝/灰蓝色系 + 星星粒子

### 🖱️ 交互功能
- 点击 ♡ 按钮收集爱心并关闭弹窗
- 可拖动弹窗位置
- 右键菜单：关闭所有、暂停/继续、关闭单个
- 实时显示收集爱心数量

### ✨ 粒子特效
- 每个主题配有独特的莫兰迪色系粒子效果
- 支持爱心、星星、花瓣等多种粒子
- 可在界面中自由开关

### 🎉 全屏结束卡片
- 弹窗结束后自动显示精美的全屏卡片
- 卡片采用莫兰迪配色，带装饰元素
- 可自定义结束语
- 点击任意位置结束程序

### 📁 自定义提示语
- 支持从 TXT 文件导入自定义提示语
- 支持在界面中添加/删除提示语
- 支持导出保存提示语文件
- 自动处理多种编码格式

### ⚡ 性能优化
- 最大窗口数量限制（默认 500）
- 使用对象池管理窗口资源

## 🚀 快速开始

### 1. 下载项目

```bash
git clone https://github.com/your-username/WarmTipPopups.git
cd WarmTipPopups
```

### 2. 运行程序

```bash
cd WarmTipPopups/WarmTipPopups
python main.py
```

### 3. 使用步骤

1. **选择主题** - 点击选择喜欢的莫兰迪主题风格
2. **管理提示语** - 可导入 TXT 文件或手动添加
3. **设置数量** - 调整弹窗数量（10-500）
4. **自定义结束语** - 输入你想要的结束语（可选）
5. **启动弹窗** - 点击"🚀 启动弹窗"按钮
6. **结束程序** - 全屏卡片出现后，点击任意位置退出

## 📝 TXT 文件格式

创建一个 `.txt` 文件，每行写一条提示语：

```txt
# 这是注释，会被忽略
愿深海的祝福与你我同在
我想看的世界，在你眼里
我心甘情愿被你困住
下次一起去看海吧
```

**注意事项：**
- 每行一条提示语
- 以 `#` 开头的行会被忽略（注释）
- 空行会被自动跳过
- 支持 UTF-8、GBK 等多种编码

## ⚙️ 配置说明

可在 `config.py` 中调整以下参数：

```python
# 弹窗配置
POPUP_WIDTH = 250          # 弹窗宽度
POPUP_HEIGHT = 60          # 弹窗高度
POPUP_MAX_COUNT = 500      # 最大弹窗数量
POPUP_ANIMATION_SPEED = 50 # 动画帧间隔(ms)
POPUP_MOVE_SPEED = 2       # 移动速度

# 粒子配置
PARTICLE_COUNT = 30        # 粒子数量
PARTICLE_FALL_SPEED = 1    # 下落速度
```

## 🔧 常见问题

### Q: 弹窗太多导致卡顿？
A: 减少弹窗数量（建议 100-200 个），或关闭粒子效果。

### Q: 如何导入自己的提示语？
A: 点击"📁 导入TXT文件"按钮，选择准备好的 TXT 文件即可。

### Q: 如何恢复默认提示语？
A: 重新运行程序即可恢复内置的 30 条提示语。

### Q: 粒子效果如何关闭？
A: 取消勾选"✨ 启用粒子效果"复选框。

### Q: 如何关闭结束卡片效果？
A: 取消勾选"🎉 显示结束卡片"复选框。

### Q: 如何自定义结束语？
A: 在控制面板的"结束语"输入框中输入你想要的文字。
